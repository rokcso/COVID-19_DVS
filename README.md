## COVID-2019 Data Visualization System

> 主要课题目标：新冠肺炎疫情传播模式分析与可视化

技术栈：

- 前端：HTML/JavaScript/CSS
- 后端：Python --> Flask
- 数据爬取：Python
- 可视化：ECharts.js
- 数据管理：MySQL
- 服务部署：云服务器（Linux 内核 / Ubuntu Server）
